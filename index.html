Backend Modes

Choose how you want to run inference

☁️ Remote Mode

Connect to a remote llama.cpp server for powerful cloud processing.

  • Use large models (70B+, 122B+)
  • Best quality with powerful models
  • Simple setup
  • Requires network connection
llama-server -hf unsloth/Qwen3.5-122B-A10B-GGUF:UD-Q4_K_XL -c 262144

📱 Local Mode

Run inference directly on your device with no internet needed.

  • Works completely offline
  • No network latency
  • Privacy - data stays on device
  • CPU / GPU / NPU acceleration
Backend: GPU0 (Adreno) | HTP0,HTP1,HTP2,HTP3 (Hexagon)

Quick Start

Get up and running in minutes

1️⃣ Download APK

Get the latest release from the GitHub releases page and install on your Android device.

2️⃣ Grant Permissions

Launch the app and grant screen capture and overlay permissions for floating controls.

3️⃣ Choose Mode

Select REMOTE (cloud) or LOCAL (on-device) mode. Configure settings as needed.

4️⃣ Start Analyzing

Tap capture to analyze your screen. Results are spoken aloud via offline TTS.
